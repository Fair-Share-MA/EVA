Uploading files:

Only Excel (any version), and CSV (any delimiter) files are accepted as inputs. Any other files will not be accepted.

Overrides:

Overrides are supplied in the form of a JSON object and are optional. This is a key -> values pair that overrides the string matching algorithm used. For example:


{
  "voter id": ["voter_id", "voter_registration_id"]
}


Here, voter id is the key and voter_id and voter_registration_id are matches.

The key is the trigger. When merging 2 files, if the algorithm comes across voter id in the first file, it will try and match it to either voter id, voter_id, or voter_registration_id in the second file. If none of these are found in the second file, the normal string matching algorithm will be used.
